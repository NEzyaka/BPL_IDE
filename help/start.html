Добро пожаловать в Turnip!

Turnip - это простой язык программирования для новичков

Для того, чтобы начать работу Вам нужно получить базовые навыки программирования на Turnip и ознакомиться с интерфейсом и полезными функциями Turnip Runner.

Полезные функции Turnip Editor:

  • Автодополнение;
  • Функция Anti-Cyrillic, которая поправит Вас, если Вы начнёте печатать оператор на русской раскладке;
  • Автоматическое сохранение каждого изменения;
  • Подсветка синтаксиса для удобного редактирования кода.
