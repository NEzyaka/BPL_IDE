Условия

Условия - это конструкция языка, обеспечивающая выполнение определённой команды
(набора команд) только при условии истинности некоторого выражения.

Синтаксис:
IF(*условие*)
*операторы, которые будут выполнены, если условие верно*
ELSE
*операторы, которые будут выполнены, если условие неверно*
ENDIF

Также внутри ELSE может быть ещё одно условие, и т.д.

Два значения можно сравнить с помощью следующих операторов:

  • == (строго равно);
  • != (не равно);
  • <= (меньше или равно);
  • >= (больше или равно);
  • < (строго меньше);
  • > (строго больше).

Также для проверки можно использовать два условия, и тогда есть два варианта случая, когда условие будет считаться верным:

  1. Верны оба условия (AND);
  2. Верно одно из них (OR).
Но в обоих случаях синтаксис условия будет таким:
IF((*условие 1*) AND / OR (*условие 2*)).
