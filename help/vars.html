Типы данных, переменные

Переменные - это ячейки в оперативной памяти компьютера, котрые хранят данные во время выполнения программ.

Для удобства программистов у каждой переменной есть свой тип данных, который позволяет хранить только определённый тип информации.

Пока в языке BPL есть 4 типа данных:

  • INTEGER (целые числа, например, 12);
  • DOUBLE (десятичные дроби, например, 15.22);
  • LINE (строки, например, "Hello");
  • BOOL (логические значения, имеет только два значения: TRUE и FALSE).

Переменная - это ячейка в памяти компьютера, у неё есть своё значение, которое занимает определённое место в оперативной памяти компьютера. Для того, чтобы работать с переменными, в BPL есть специальные операторы для работы с ними, их 13:

  • PRINT (В данном случае: вывод на экран значения переменной);
  • INPUTVAR (Ввод значения пееменной с клавиатуры);
  • DUMP (Вывод на экран информации обо всех объявленных переменных);
  • DUMPVAR (Вывод на экран информации об одной иди двух переменных);
  • SWAP (Замена местами значений двух переменных);
  • DELETE (Удаление переменной из памяти);
  • SIZE (Вывод на экран размера переменной в памяти);
  • SETVAL (Установка точного значения переменной);
  • INVERT (Инвертирование значения переменной);
  • ADD (Прибавление к значению какой-либо переменной числа, или значения переменной);
  • SUBTRACT (Вычитание из значения какой-либо переменной числа, или значения переменной);
  • MULTIPLY (Умножение значения какой-либо переменной на число, или значение переменной);
  • DIVISE (Деление значения какой-либо переменной на число, или значение переменной).

Для того, чтобы объявить переменную нужно сделать следующую запись: "ТИП ДАННЫХ имя переменной"

Давайте разберём пример работы с переменными. Например, нам нужно узнать имя пользователя и вывести 'Hello %введённое имя пользователя%' Для этого нам нужно: звести переменную строкового типа, считать её значение, вывести 'Hello ' + значение переменной. Должен получиться следующий код:
LINE name
INPUTVAR name
PRINT "Hello " AND name

Также можно иинициализировать (то есть, задать значение сразу, при объявлении). Это не сложнее, чем просто объявить переменную: "ТИП ДАННЫХ имя переменной = значение".

Можно привести ещё пример: замена значения переменной. Нам необходимо завести переменную с каким-то значением, а потом поменять это значение на какое-либо другое. В качестве проверки значения будем использовать оператор DUMPVAR. Код:
INTEGER a = 15
DUMPVAR a
SETVAL a 20
DUMPVAR a
