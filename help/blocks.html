Блоки

Блоки - это разделение кода на определённые участки для удобства программиста.

В блоке находятся инструкции, которые будут выполнены тогда, когда этот блок будет вызван.

Блок вызывается с помощью оператора doblock. Например,"doblock myFirstBlock".

Выполнение программы всегда начинается с блока start. Начало блока объявляется с помощью оператора beginblock, а конец - с помощью endblock.

Давайте рассмотрим пример блока:
beginblock start
print "Hello, world!"
endblock

Синтаксис:
beginblock имя блока
*содержиме блока*
endblock

Также можно импортировать блоки из других файлов, это делает оператор IMPORT. Например, "import C:\test.txt".

Ну и напоследок, КАЖДЫЙ оператор, кроме import должен находиться в блоке, все операторы, кроме import, которые находятся вне какого-либо блока, выполняться не будут.
