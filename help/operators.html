Операторы

Оператор - это команда, которую получает, обрабатывает и выполняет специальная программа - интерпретатор.

Интерпретатор - это специальная программа, которая выполняет команды, которые поступают из кода.

Синатксис:

ОПЕРАТОР аргумент
или просто
ОПЕРАТОР

Пока в языке BPL есть 21 оператор:

  • PRINT;
  • INPUTVAR;
  • ALERT;
  • NEXTLINE;
  • CLEARSCREEN;
  • DUMP;
  • DUMPVAR;
  • COMMAND;
  • FONTCOLOR;
  • SWAP;
  • DELETE;
  • SIZE;
  • SETVAL;
  • DOBLOCK;
  • BLOCKLIST;
  • IMPORT;
  • INVERT;
  • ADD;
  • SUBTRACT;
  • MULTIPLY;
  • DIVISE.

Возможно, по их названиям Вы уже поняли, что делает каждый из представленных операторов, но всё равно следует подробно разобрать каждый из них.

Оператор PRINT выводит в консоль строку/значение переменной/и то, и другое. Например, команда "PRINT "Hello, world!"" выведет в консоль строку Hello, world!

Оператор ALERT издаёт звуковой сигнал.

Оператор NEXTLINE переводит строку на следующую. Например программа
PRINT "First Line"
NEXTLINE
PRINT "Second Line"
выведет в консоль
First Line
Second Line

Оператор CLEARSCREEN очищает консоль.

Оператор COMMAND запускает командную строку с определёнными аргументами. Например, оператор "COMMAND help" откроет окно командной строки, где будет показан справка по командам.

Оператор FONTCOLOR устанавливает определённый цвет шрифта. Например, "FONTCOLOR LIGHTGREEN" установит ярко-зелёный цвет шрифта в консоли.

Так как остальные операторы работают с переменными, их работа будет описана в разделе 'Типы данных'
