Типы данных, переменные величины

Переменные - это ячейки в оперативной памяти компьютера, котрые хранят данные во время выполнения программ.

Для удобства программистов у каждой переменной есть свой тип данных, который позволяет хранить только определённый тип информации.

Пока в языке BPL есть 4 типа данных:
INTEGER (целые числа, например, 12)
DOUBLE (десятичные дроби, например, 15.22)
LINE (строки, например, Hello)
BOOL (логические значения, имеет только два значения: TRUE и FALSE).

Переменная - это ячейка в памяти компьютера, у неё есть своё значение, которое занимает определённое место в оперативной памяти компьютера. Для того, чтобы работать с переменными, в BPL есть специальные операторы для работы с ними, их 6:
DUMP (Вывод на экран информации о всех переменных: Номер переменной: Имя -> Значение [Тип данных, Размер в байтах]);
DUMPVAR (Вывод на экран информации о переменной: Имя -> Значение[Тип данных, Размер в байтах]);
SWAP (Замена местами значений двух переменных);
SETVAL (Установка значения переменной);
DELETE (Удаление переменной);
INPUTVAR (Ввод значения пееменной с клавиатуры).

Давайте разберём пример работы с переменными. Например, нам нужно узнать имя пользователя и вывести 'Hello %введённое имя пользователя%' Для этого нам нужно: звести переменную строкового типа, считать её значение, вывести 'Hello ' + значение переменной.
Должен получиться следующий код:


LINE name
INPUTVAR name
PRINT "Hello ", name

Можно привести ещё пример: замена значения переменной.
Нам необходимо завести переменную с каким-то значением, а потом поменять это значение на какое-либо другое. В качестве проверки значения будем использовать оператор DUMPVAR.
Код:


INTEGER int = 15
DUMPVAR a
SETVAL a 20
DUMPVAR a
