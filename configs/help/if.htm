Условия

Условия - это конструкция языка, обеспечивающая выполнение определённой команды
(набора команд) только при условии истинности некоторого выражения.

Синтаксис:

IF(*условие*)
*операторы, которые будут выполнены, если условие верно*
ELSE
*операторы, которые будут выполнены, если условие неверно*
ENDIF

Также внутри ELSE может быть ещё одно условие, и т.д.

Условия - это очень полезно, например, напишем программу для проверки пароля:
LINE password = qwerty
LINE inputPassword
INPUTVAR inputPassword
IF(inputPassword == password)
PRINT "Welcome back!"
ELSE
PRINT "Wrong password!"
ALERT
ENDIF

Усложним нашу программу, добавив проверку имени пользователя, для этого нам понадобится вложенное условие, то есть, условие в условии:
LINE password = qwerty
LINE inputPassword
LINE username = root
LINE inputUsername
INPUTVAR inputUsername
IF(inputUsername == username)
INPUTVAR inputPassword
IF(inputPassword == password)
PRINT "Welcome back!"
ELSE
PRINT "Wrong password!"
ALERT
ENDIF
ELSE
PRINT "Wrong username!"
ALERT
ENDIF
