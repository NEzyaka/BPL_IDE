Операторы

Оператор - это инструкция, команда, которую получает, обрабатывает и выполняет специальная программа - интерпретатор.

Интерпретатор - это специальная программа, которая выполняет команды, которые поступают из кода.

Пока в языке BPL есть 10 операторов:
PRINT;
DUMP;
DUMPVAR;
NEXTLINE;
COMMAND;
FONTCOLOR;
SWAP;
SETVAL;
DELETE;
ALERT.

Возможно, по их названиям Вы уже поняли, что делает каждый из представленных операторов, но всё равно следует подробно разобрать каждый из них.

Оператор PRINT выводит в консоль строку. Например,
PRINT "Hello, world!" выведет в консоль Hello, world!

Оператор ALERT издаёт звуковой сигнал.

Оператор COMMAND запускает командную строку с определёнными аргументами. Например, оператор
COMMAND help откроет окно командной строки, где будет показан справка по командам.

Оператор FONTCOLOR устанавливает определённый цвет шрифта. Например,
FONTCOLOR LIGHTGREEN установит ярко-зелёный цвет шрифта в консоли.

Оператор NEXTLINE переводит строку на следующую. Например программа
PRINT "First Line"
NEXTLINE
PRINT "Second line"
выведет в консоль
First Line
Second Line

Так как остальные операторы работают с переменными, их работа будет описана в разделе 'Типы данных'
