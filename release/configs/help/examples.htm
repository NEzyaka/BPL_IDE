Примеры

Hello, world!:
PRINT "Hello, world!"

Консоль, как в матрице:
FONTCOLOR LIGHTGREEN

Hello %username%:
LINE name
INPUTVAR name
PRINT "Hello ", name

Перевод строк:
PRINT "First Line"
NEXTLINE
PRINT "Second line"

Программа для проверки пароля:
LINE password = qwerty
LINE inputPassword
INPUTVAR inputPassword
IF(inputPassword == password)
PRINT "Welcome back!"
ELSE
PRINT "Wrong password!"
ALERT
ENDIF
